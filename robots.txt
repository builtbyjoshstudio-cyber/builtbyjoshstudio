# builtbyjoshstudio.com — robots.txt
#
# Strategy
# --------
#   1. Default-allow for all crawlers
#   2. Block infrastructure paths from everyone (/cdn-cgi/, *.json)
#   3. Block AI TRAINING crawlers from JS/CSS plumbing (/js/, /css/)
#      — they get the content, not the build artifacts
#   4. Search engines and AI LIVE-RETRIEVAL crawlers keep full access
#      so page rendering works correctly for indexing and citations
#
# Bot categories
# --------------
#   TRAINING       (no /js/ /css/): GPTBot, ClaudeBot, anthropic-ai,
#                                   Google-Extended, Applebot-Extended,
#                                   meta-externalagent, CCBot
#   LIVE RETRIEVAL (full access):  ChatGPT-User, OAI-SearchBot, Claude-Web,
#                                   PerplexityBot, Perplexity-User
#   SEARCH ENGINES (full access):  Googlebot, Bingbot, BingPreview, Applebot,
#                                   DuckDuckBot, YouBot, FacebookExternalHit
